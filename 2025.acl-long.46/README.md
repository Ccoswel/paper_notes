# MIND 论文总结报告

## ACL 2025: MIND - 零样本有害 Meme 检测的多智能体框架

### 核心贡献

北京邮电大学团队在 ACL 2025 提出的 MIND 框架，解决了有害 Meme 检测中的两大挑战：**标注数据稀缺**和**领域迁移困难**。MIND 是首个将多智能体系统引入该领域的零样本方法。

### 技术架构

MIND 由三个模块组成：
1. **SSR（相似样本检索）**：通过多模态嵌入（视觉+文本编码器）和余弦相似度，从历史数据中检索与目标 Meme 最相似的 K 个样本
2. **RID（相关洞察推导）**：采用双向推导机制——前向推导从部分样本归纳通用规律，后向推导从完整集合精炼洞察
3. **IAI（洞察增强推理）**：融合双向推导结果，对目标 Meme 进行最终分类

### 实验结果

在 HarM、FHM、MAMI 三个基准数据集上，MIND 大幅超越 GPT-4V、LLaVA-1.5 等基线模型，宏观 F1 分数提升 7.7%-21.0%。消融实验验证了各模块的必要性。

### 意义与启示

MIND 证明了**无需标注数据**的多智能体协作可实现高效有害内容检测，为内容审核提供了低成本、高泛化的新范式。代码已开源：https://github.com/destroy-lonely/MIND
